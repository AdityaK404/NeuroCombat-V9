# Example MMA Fight Videos

Place your MMA fight video files in this directory for processing.

## Supported Formats
- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)

## Recommended Specifications
- Resolution: 720p or higher
- FPS: 24-30
- Duration: 1-10 minutes for testing
- Clear view of both fighters
- Good lighting conditions

## Sample Videos
You can test the system with:
1. YouTube MMA fight clips (download and place here)
2. Personal training footage
3. Amateur fight recordings

## Example Structure
```
examples/
├── fight_clip_1.mp4
├── fight_clip_2.mp4
└── training_session.mp4
```
